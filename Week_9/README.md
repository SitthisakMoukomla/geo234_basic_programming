## ตัวอย่างการประยุกต์ใช้ทางภูมิศาสตร์

- **DEM (Digital Elevation Model):** จำลองข้อมูลความสูงเป็น 2D Array และแสดงผลเป็นแผนที่
- **Slope:** คำนวณความชันจาก DEM ด้วย `np.gradient()`
- **NDVI:** คำนวณดัชนีพืชพรรณจากข้อมูล Band Red และ NIR พร้อมจำแนกประเภทพื้นที่
- **PM2.5:** วิเคราะห์ข้อมูลคุณภาพอากาศหลายจังหวัดด้วย 2D Array และสถิติรายแกน

## ไลบรารีที่ใช้

- `numpy` — การคำนวณเชิงตัวเลขและจัดการอาร์เรย์หลายมิติ
- `matplotlib` — การแสดงผลข้อมูลเชิงพื้นที่ (DEM, Slope, NDVI)
- `scipy.ndimage` — Gaussian filter สำหรับทำให้ข้อมูล DEM เรียบขึ้น
